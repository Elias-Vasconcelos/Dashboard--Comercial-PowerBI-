# 📊 Dashboard Comercial - Power BI

## 📌 Sobre

Este projeto é um **dashboard comercial** desenvolvido no **Power BI**, utilizando dados de vendas para gerar insights de forma visual e interativa.

---

## 🗂️ Dados

Os dados foram extraídos de planilhas Excel com as seguintes bases:

* Base Vendas
* Cadastro de Clientes
* Cadastro de Lojas
* Cadastro de Produtos

---

## 📈 O que o dashboard mostra

* 💰 Receita total
* 📦 Produtos vendidos
* 🧾 Ticket médio
* 👥 Clientes ativos
* 📉 Vendas ao longo do tempo
* 🌎 Vendas por região
* 🏷️ Vendas por marca

---

## ⚙️ Tecnologias

* Power BI
* Excel
* DAX
